Autoexam Generador y evaluador minimalista de cuestionarios

Autoexam es un generador y evaluador automático de cuestionarios (exámenes, encuestas, votaciones, etc.). Para comenzar, elije el tipo de proyecto que deseas:

Exámenes

Los exámenes se componen de un conjunto de preguntas con opciones de selección, y una clave que determina el valor de cada pregunta.

Crear Examen

Encuestas

Las encuestas se componen de un conjunto de preguntas con opciones de selección. A diferencia de los exámenes, no existe el concepto de clave.

Crear Encuesta

Boletas

Las boletas consisten en una sola pregunta con un conjunto de opciones de selección. Se puede elegir una, varias o todas las opciones.

Crear Boleta

Abrir proyecto existente

Si tienes un proyecto de Autoexam, sube el archivo con extensión .apr para editarlo.
